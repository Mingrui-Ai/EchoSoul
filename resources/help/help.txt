EchoSoul 使用指南

一、主界面介绍
主界面分为四个主要区域，方便您快速上手：
1. 顶部工具区：显示应用标题和窗口控制按钮（最小化、关闭）。
2. 左侧列表区：包括会话列表和联系人列表，支持置顶和搜索。
3. 右侧会话区：展示对话内容，包括消息气泡、时间分隔线，支持图片预览和复制操作。
4. 底部输入区：包含多行输入框和发送按钮，可选自动播放语音开关。

二、会话与消息
1. 新建会话：在左侧点击“新建”或从联系人列表发起，系统会自动生成对话标题。
2. 发送消息：输入内容后按回车发送；需要换行时，请使用 Shift + Enter。
3. 复制与引用：将鼠标悬停在消息气泡上，可选择复制或引用该消息。
4. 图片与文件：
   - 支持将图片直接粘贴到输入框上传。
   - 也可将图片拖拽至输入框添加。
   - 点击消息中的图片可放大预览，再次点击关闭。
5. 清空会话：在会话头部菜单中选择“清空”，该操作仅影响当前会话。
6. 删除消息：选中单条或多条消息后可删除，删除后无法恢复，请谨慎操作。

三、联系人管理
1. 新建联系人：在联系人面板中添加，填写名称和备注（可选）。
2. 备注与置顶：可为联系人添加备注，或将常用联系人置顶显示。
3. 发起会话：点击联系人项即可开始聊天。
4. 搜索联系人：在搜索框中输入关键字，快速筛选联系人。

四、历史记录
1. 自动保存：可在设置中开启或关闭自动保存功能，默认保存至 ./history 目录。
2. 手动保存：在会话菜单中点击“保存”，将当前会话保存至历史目录。
3. 上限策略：可设置单会话消息条数和会话总数上限，超出时将自动清理早期记录。
4. 导入与导出：支持从历史目录导入会话，或将会话导出为可分享的文件。
5. 清理历史：在历史视图中可按条件批量删除旧记录。

五、搜索与筛选
1. 会话列表筛选：输入关键字，快速查找会话标题或相关联系人。
2. 消息内容搜索：在当前会话中搜索关键字，快速定位并跳转至匹配消息。
3. 条件筛选：支持按时间范围或是否包含图片等条件进行筛选。

六、设置中心
入口位于“设置 > 用户设置”，常用设置包括：
1. 用户设置：
   - 头像：点击头像可选择本地图片（支持 PNG、JPG、WEBP格式），选择后预览并保存。
   - 重置头像：点击“重置”可恢复为默认头像。
2. 接口与网络：
   - 配置 DeepSeek 与百度接口参数。
   - 设置连接超时与读取超时时间（单位：秒）。
3. 历史策略：
   - 开启或关闭自动保存，设置保存路径。
   - 设置单会话消息条数和会话数量上限。
4. 生成参数：
   - 调整生成温度（temperature）和最大生成长度（max_tokens）。
5. 外观与布局：
   - 设置侧边栏是否折叠、界面缩放比例和字体大小。
6. 语音设置：
   - 开启或关闭语音自动播放，选择播放设备。

所有设置将保存至 config.properties 文件，部分设置需重启生效。

七、更换头像步骤
1. 进入“设置 > 用户设置”。
2. 点击当前头像，从本地选择图片。
3. 选中后，缩略图显示高亮边框作为反馈。
4. 在预览区确认效果，点击“保存”。
5. 返回主界面，顶部和消息中的头像将同步更新。如更新失败，请检查图片是否可读或路径是否有效。

八、应用图标
1. 窗口图标：将 PNG 格式图标放置于 src/main/resources/images/app_icon.png，启动后自动显示为窗口和任务栏图标。
2. 安装包图标：准备 ICO 格式图标，存放于 src/main/resources/images/app_icon.ico，打包时使用 --icon 参数指定。如未生效，请检查资源是否已正确包含。

九、窗口与布局
1. 侧边栏：点击折叠按钮或拖拽分割线调整宽度。
2. 消息区：支持选中文本复制，点击图片可放大预览。
3. 输入区：支持多行编辑和快捷清空。

十、数据迁移与备份
1. 迁移：将 config.properties 文件和 ./history 目录复制到新环境后启动应用。
2. 备份：建议定期备份 ./history 目录，防止数据丢失。
3. 重置：删除或重命名 config.properties 文件，可恢复默认设置。

十一、常见问题
1. 头像未更新：请检查图片路径是否正确、格式是否受支持，必要时重启应用。
2. 图标未生效：确认资源路径正确，重新打包应用，或尝试清理系统图标缓存。
3. 请求失败：检查接口地址和密钥是否正确，确认网络连接正常，或适当增加超时时间。
4. 历史未保存：请确认已开启自动保存或手动保存，并确保保存目录具有写入权限。
